# Discord bot с городской валютой

Бот создает городскую экономику внутри Discord:

- пользователи зарабатывают валюту за работу на благо города;
- берут и сдают разные задания;
- покупают товары в магазине;
- отправляют заявки на вывод валюты в рубли;
- администрация управляет товарами, заданиями и заявками на вывод.

## Запуск

1. Установить зависимости:

```bash
pip install -r requirements.txt
```

2. Положить токен в переменную окружения `DISCORD_TOKEN` или в файл `discord token.txt`.

3. Запустить:

```bash
python bot.py
```

## Команды пользователей

- `!помощь`
- `!баланс`
- `!работа`
- `!задания`
- `!взять <id>`
- `!сдать <id>`
- `!магазин`
- `!купить <id> [кол-во]`
- `!инвентарь`
- `!курс`
- `!вывести <сумма>`

## Команды администрации

- `!добавитьтовар <цена> <название> | <описание>`
- `!удалитьтовар <id>`
- `!добавитьзадание <мин> <макс> <название> | <описание>`
- `!удалитьзадание <id>`
- `!выводы`
- `!одобритьвывод <id>`
- `!отклонитьвывод <id>`

## Как работает вывод

Команда `!вывести` не переводит деньги автоматически. Она создает заявку в базе, а администрация уже отдельно обрабатывает ее и подтверждает выплату в рублях. Это безопаснее и удобнее для ручной проверки.
